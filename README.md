# `byte-table`

High-performance 256-length array indexed by `u8`. Uses `#![no_std]`.

## Usage

To use `byte-table`, first add this to your `Cargo.toml`:

```toml
[dependencies]
byte-table = "1"
```

Next, add this to your crate:

```rust
use byte_table::ByteTable;
```
